# beetlebot PRD (Normalized)

beetlebot is a chat-first life-operations agent that proactively plans social time, travel experiences, and personal free-time activities.

## MVP Must-Haves

- Chat interface (WhatsApp-first concept, web console for control).
- Modes and context-aware planning.
- Calendar + weather integration surfaces.
- Activity suggestions + soft-hold scheduling.
- Approval flow before spending.
- Basic group coordination.
- Weekly planning loop.

## Product Language

- Cron jobs are presented as **Autopilots**.
- Skills are presented as **Packs**.
- Core promise: surprise with consent.
